Awal mula saya mulai belajar tentang programming yaitu pada tahun pertama kuliah, yang dimana pada salah satu matakuliah yang saya program mewajibkan mahasiswa untuk mengerti tentang perkodingan dasar tentang HTML, CSS dan sebagainya.

Setelah melewati beberapa semester saya mulai tertarik dengan bahasa programming Java untuk saya tekuni lebih dalam, dan saya sudah 3 tahun lebih belajar menggunakan bahasa ini.
